Objective 2: PSEUDOCODE

Task 3: Create a small calculator which only does ‘+’ or ‘-‘Operations. (Hint: Take three variable inputs with
one being used for the operator)

INPUT Num1
INPUT Num2
INPUT Operator

SET Result to 0

IF Operator = '+' THEN
   Result = Num1 + Num2
ELSE
   Result = Num1 - Num2
ENDIF

OUTPUT Result
